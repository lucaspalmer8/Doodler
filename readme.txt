All requirements have been implemented.
The text files are saved with extension ".txt".
The binary files are saved with extension ".ser" (for serializable).
If the "All files" of the ".txt" filter is selected in the save file dialog, then the file will be saved as a text file.
If the ".ser" filter is selected in the save file dialog, then the file will be saved as a binary file.

Addition enhancements:
1.) Forward and reverse animation implemented.
2.) Accurate representation of time implemented.
